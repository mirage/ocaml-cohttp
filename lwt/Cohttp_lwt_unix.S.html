Module type Cohttp_lwt_unix.S

module type S = sig .. end
This module type defines the additional UNIX-specific functions that are exposed in addition to the Cohttp_lwt.Server interface. These are primarily filesystem functions, and also Cohttp_lwt_unix.S.create to actually bind the server to a socket and respond to incoming requests.
include Cohttp_lwt.Server

       ?mode:Conduit_lwt_unix.server -> t -> unit Lwt.t
